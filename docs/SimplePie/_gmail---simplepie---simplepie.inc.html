SimplePie

A PHP-Based RSS and Atom Feed Framework. Takes the hard work out of managing a complete RSS/Atom solution.

Copyright (c) 2004-2008, Ryan Parman and Geoffrey Sneddon All rights reserved.

Modified 2008 by Jason Meinzer to support HTTP proxies (cURL mode only) Define your proxy hostname/port as USE_PROXY e.g. define(USE_PROXY, 'proxy:port') Define your proxy credentials as USE_PROXYAUTH e.g. define(USE_PROXYAUTH, 'username:password')
